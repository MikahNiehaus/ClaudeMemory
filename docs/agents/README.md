# Agents

13 specialist agents for different software engineering domains.

## Agent List

| Agent | Expertise | When to Use |
|-------|-----------|-------------|
| [test-agent](./test-agent.md) | Testing, TDD, coverage | Writing tests, test strategy |
| [debug-agent](./debug-agent.md) | Bug analysis, root cause | Errors, debugging |
| [architect-agent](./architect-agent.md) | Design, SOLID, patterns | Architecture decisions |
| [reviewer-agent](./reviewer-agent.md) | PR review, feedback | Code reviews |
| [docs-agent](./docs-agent.md) | Documentation | Writing docs |
| [estimator-agent](./estimator-agent.md) | Story points | Ticket estimation |
| [ui-agent](./ui-agent.md) | UI implementation | Frontend, mockups |
| [workflow-agent](./workflow-agent.md) | Execution, process | Complex implementations |
| [research-agent](./research-agent.md) | Web research | Deep research, fact-checking |
| [security-agent](./security-agent.md) | Security review, OWASP | Security audits |
| [refactor-agent](./refactor-agent.md) | Code smells | Code cleanup, tech debt |
| [explore-agent](./explore-agent.md) | Codebase exploration | Understanding code |
| [performance-agent](./performance-agent.md) | Profiling, optimization | Performance issues |

## Quick Decision Guide

```
Need to...                     → Use
────────────────────────────────────────
Write or review tests          → test-agent
Fix a bug or error             → debug-agent
Design system architecture     → architect-agent
Review a pull request          → reviewer-agent
Write documentation            → docs-agent
Estimate a ticket/story        → estimator-agent
Implement UI from mockup       → ui-agent
Plan complex implementation    → workflow-agent
Research external topics       → research-agent
Security audit or review       → security-agent
Clean up / refactor code       → refactor-agent
Understand codebase            → explore-agent
Optimize performance           → performance-agent
```
